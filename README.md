# Circle Zhou - Personal Website

[Visit the site here.](http://CircleZ3791117.github.io/)

Tested with:
- Node v8.0.0
- NPM v5.0.0
- Bundler v1.16.0
- Gem v2.7.1

If you're using NVM, this will get you to the right Node/NPM versions:
```$bash
nvm use v8.0.0
```

To run:
```$bash
bundle install
npm install
bundle exec gulp
```
